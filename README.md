# luma-mcp-tools

MCP tool servers for AI agents.

## Servers

### GitPulse
Workspace-aware git repo scanning, diagnosis, and safe sync. 6 tools for discovering repos, health checks, safe pulls, and fork sync.

### Obsidian Memory
Vault-based persistent knowledge graph with wikilinks, backlinks, and multi-vault support. Backward-compatible with flat-file memory tiers.

## Install

```bash
git clone https://github.com/ecuadralmft/luma-mcp-tools.git
cd luma-mcp-tools
./install.sh
```

The installer creates Python venvs, installs dependencies, and generates your `~/.kiro/settings/mcp.json`.
